ReadMe for Dublin Bike Helper

To build a website and learn python

Basic Idea
To calculate the route and time required to travel between two points in Dublin city using Dublin bike’s and walking 
Calculate the walking route and time required to the nearest bike station from your start location
Show availability of bikes at the nearest bike station
Calculate the cycling route and time required to the nearest bike station from destination point
Show availability to return bike 
Show timer and any cost
Calculate the walking route and time to destination from nearest bike station
Display results 
Options 
Use map to display the results
Show audio or visual results as you travel along the route
